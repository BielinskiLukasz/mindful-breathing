## Overview

v1.0 delivered four targeted improvements to a fully-functional single-file breathing app. The milestone closed critical UX gaps discovered after v0.6 shipped: a media query bug that broke compact landscape layout on tall phones, abrupt background color jumps on phase changes in light mode, missing incomplete session tracking in history, and no version indicator in the info panel.

All changes are isolated to `index.html`. No new dependencies, no architecture changes. The "Pause" rename also correctly labels the stop button to match actual app behavior.

## Milestone Summary

**Key Decisions:**

- `background` shorthand (not `background-color`) in body CSS transition — animates CSS custom property `--bg` changes reliably; `background-color` is a sub-property that doesn't track the shorthand (VISUAL-01, D-10, D-13)
- `height: auto` + `min-height: 100vh` in compact landscape body — overrides base `height: 100vh` so content can scroll past viewport while preserving centering for short content (LAYOUT-01, D-14, D-15)
- `APP_VERSION` JS-injected at init, HTML element starts empty — CONFIG constant is single source of truth; no hardcoded value in markup (INFO-01, D-16)
- `cycleCount >= 2` threshold for incomplete save — cycleCount initializes to 1 and increments after each full cycle; `>= 2` means at least 1 complete cycle done (TRACK-01, D-01)
- Incomplete save fires in toggleBtn click handler only, not in `stop()` or `reset()` — prevents duplicate entries on pause/resume cycles (TRACK-01, D-02)
- "Pause" replaces "Stop" in `UI.btnStop` — reflects actual behavior; Resume/Start appears after stopping (UX fix)

**Issues Resolved:**

- Compact landscape body height clamped by `height: 100vh` — breathing ring/Start button cut off on iPhone 14 Pro Max landscape
- Background phase tint jumping abruptly in light mode — CSS custom property `--bg` not animated via `background-color` sub-property
- No record of sessions the user abandoned mid-session — history only showed completed sessions

**Issues Deferred:**

- CSV import — RFC 4180 edge cases deferred to v1.1+
- Full accessibility (keyboard nav, ARIA labels, focus indicators) — deferred to v1.1+

**Technical Debt Incurred:**

None — all 4 requirements fully implemented with no placeholder values or stubs.
