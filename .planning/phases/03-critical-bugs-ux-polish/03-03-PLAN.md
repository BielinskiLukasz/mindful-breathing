```css
  0%   { filter: brightness(1); }
```

**Location 2 — Ring flash trigger (in advanceToNextPhase(), after cueOnPhaseEnter() call at line ~1734):**

```javascript
// Trigger ring flash animation on phase transition
ringProgressEl.classList.remove("ringFlash");
```

**Why this approach:**
- Brightness filter (not opacity or transform) preserves ring visibility and doesn't interfere with SVG stroke rendering
- 200ms duration matches existing countdown pop animation timing
- Reflow trigger (offsetWidth read) ensures animation reruns even on consecutive phase advances
- No performance cost; filter runs on GPU

**Theme compatibility:**
- Dark theme: Ring brightens from #9aa0a6 to ~1.3× brightness; noticeable against dark background
- Light theme: Ring brightens from #d4a574 to ~1.3× brightness; visible on light card background

**Location 1 — HTML gesture hint element (add before Start/Stop buttons container, around line ~xxx):**

```html
<div id="gestureHint" style="opacity: 0; transition: opacity 300ms ease; margin-bottom: 12px; padding: 8px 12px; background: rgba(52,211,153,0.15); border-radius: 8px; border: 1px solid rgba(52,211,153,0.3); font-size: 13px; color: var(--accent); text-align: center;">
```

**Location 2 — Hint display function (add in script section around line ~2100, before or after existing helper functions):**

```javascript
function showGestureHint() {
  const hintEl = document.getElementById("gestureHint");
  if (!hintEl) return;
  
  hintEl.style.opacity = "1";
  
  // Hide hint after first interaction (click or keydown anywhere)
  const hideHint = () => {
    hintEl.style.opacity = "0";
    localStorage.setItem("gestureHintShown", "true");
    document.removeEventListener("click", hideHint);
    document.removeEventListener("keydown", hideHint);
  };
  
  document.addEventListener("click", hideHint);
  document.addEventListener("keydown", hideHint);
}
```

**Location 3 — Initialization check (in the app startup section around line ~2152, with other loadSettings/render calls):**

Add before other initialization:
```javascript
// Show gesture hint on first visit
if (!localStorage.getItem("gestureHintShown")) {
  showGestureHint();
}
```

**Why this approach:**
- localStorage.getItem("gestureHintShown") tracks truly-first visit across all tabs/windows of the origin
- Opacity transition (CSS) provides smooth fade-in/fade-out without animation overhead
- Event delegation hides hint on ANY click or keydown (Start button, Space key, preset click, etc.)
- Non-intrusive; green accent color matches UI palette; positioned above controls where it doesn't block timer display
