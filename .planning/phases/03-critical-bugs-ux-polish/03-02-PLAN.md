  <action>
Enhance CSS `:active` pseudo-class for buttons to provide stronger press-down visual feedback.

**Location:** CSS button styles around line 204 (existing CSS rule `button:active`)

**Current CSS (line ~204):**
```css
button:active { transform: scale(0.97); }
```

**Enhanced CSS:** Replace with:
```css
button:active {
  transform: scale(0.97);
  background: rgba(255,255,255,0.1);
  box-shadow: inset 0 2px 4px rgba(0,0,0,0.3);
}
```

**Rationale:**
- `transform: scale(0.97)` — Existing shrink-down effect (keep)
- `background: rgba(255,255,255,0.1)` — Subtle light overlay on press (works on both dark and light themes)
- `box-shadow: inset 0 2px 4px rgba(0,0,0,0.3)` — Inset shadow creates embossed "pressed-in" effect

**Why CSS-only:** No JavaScript needed. CSS `:active` pseudo-class is native browser feature; no event overhead.

**Theme compatibility:**
- Dark theme: White overlay and dark inset shadow both visible on dark background
- Light theme: White overlay is subtle; dark inset shadow still works against light card background
- Both themes benefit from inset shadow creating depth

Per MICRO-01.
